# Highrise Bot

ربات هایرایز با قابلیت‌های کامل: دنس لوپ، تیپ، مدیریت روم، تله‌پورت، گفتگو فارسی/انگلیسی.

## Run & Operate

- `cd bot && python main.py` — اجرای مستقیم بات
- Workflow: **Highrise Bot** — پروسه خودکار بات

## Required Secrets & Env Vars

- `HIGHRISE_TOKEN` — توکن بات (در Replit Secrets)
- `HIGHRISE_ROOM_ID` — آیدی روم (در Replit Env Vars) = `69973d1f53b9dad3992371a6`

## Stack

- Python 3.11
- highrise-bot-sdk >= 24.1.0
- asyncio

## Where things live

- `bot/main.py` — entry point
- `bot/bot.py` — کلاس اصلی بات با تمام رویدادها و دستورها
- `bot/dances.py` — لیست ۳۰۰+ دنس + تبدیل اعداد فارسی
- `bot/responses.py` — پاسخ‌های فارسی/انگلیسی، جوک، خوش‌وبش
- `bot/config.py` — لود/سیو ادمین‌ها و تله‌پورت از فایل JSON
- `bot/admins.json` — لیست ادمین‌های بات (تولید میشه)
- `bot/teleport.json` — موقعیت تله‌پورت (تولید میشه)

## Bot Commands

### همه کاربران
| دستور | عملکرد |
|-------|--------|
| عدد ۱ تا ۳۱۲ (فارسی یا انگلیسی) | شروع دنس در لوپ |
| `stop` | توقف دنس |
| `help` / `راهنما` | راهنما (پیام خصوصی) |
| سلام، خوبی، جوک، بای... | گفتگوی فارسی/انگلیسی |

### فقط ادمین بات + صاحب روم
| دستور | عملکرد |
|-------|--------|
| `tip all` | تیپ به همه |
| `tip @username` | تیپ به یک نفر |
| `kick @user` / `kick@user` | اخراج |
| `mute @user [ثانیه]` | میوت |
| `ban @user` | بن |
| `unban @user` | آن‌بن |
| `set tel` | ذخیره موقعیت تله‌پورت |
| `unset tele` | حذف تله‌پورت |

### فقط صاحب روم
| دستور | عملکرد |
|-------|--------|
| `admin add @user` | اضافه کردن ادمین بات |
| `admin remove @user` | حذف ادمین بات |
| `admin list` | لیست ادمین‌ها |
| `بیا بات` / `come here bot` | بات میاد پیشت |

### ویپر خصوصی
| دستور | عملکرد |
|-------|--------|
| `tp` / `tele` / `teleport` | تله‌پورت به موقعیت ذخیره‌شده |

## Auto Behaviors

- ورود کاربر → پیام خوش‌آمد خصوصی + دنس تصادفی بات
- خروج کاربر → پیام خداحافظی عمومی در روم
- بات آنلاین → پیام اطلاع‌رسانی در روم

## Architecture decisions

- ادمین‌ها در `admins.json` ذخیره میشن (بین ری‌استارت‌ها حفظ میشه)
- دنس لوپ با `asyncio.Task` مدیریت میشه — هر کاربر task جداگانه دارد
- owner_id از `session_metadata` در `on_start` دریافت میشه
- اعداد فارسی با دیکشنری تبدیل میشن

## Gotchas

- Room ID باید `ownedRoomId` از URL باشه نه `id`
- اگه بات کیک شد، خودکار دوباره وصل میشه (SDK داخلی)
- دنس‌های لیست شده ID هایرایز هستن — اگه ID وجود نداشت SDK ارور میده ولی بات ادامه میده

## User preferences

- فارسی و انگلیسی هر دو پشتیبانی بشن
